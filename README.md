## ✨ Features
* **Peer-to-Peer Co-op:** Join friends via Steam or Direct IP.
* **Shared Progression:** World state saves for the host.
* **Synced Mechanics:** Building, hunting, and weather are fully synced.

---

## 🛠 Installation
1. Download
2. Extract the contents into your `Mad Island` Bepeinex\plugins folder.
3. Launch the game and hit F8 then look for the **"Multiplayer"** button in the main menu.
4. It will ask for IP either host/join
